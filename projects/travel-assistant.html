Agentic Travel Assistant

Conversational Flight Booking with Real-Time Fares

OpenAI API Python Flight APIs NLP
Agentic Travel

Overview

Booking a flight is still frustratingly transactional. You navigate to a search engine, fill out a form, filter through hundreds of results, and repeat for each leg. This project builds a conversational AI agent that handles the entire booking flow through natural language — you just describe your trip.

Say something like: "I need to get from San Francisco to Austin on the 15th, coming back the 19th, and I want to avoid layovers if it's under $400" — and the agent handles the rest.

Architecture

The assistant is built using OpenAI's function-calling API, which lets the model decide when to invoke real tools. The agent has access to several tools:

  • search_flights: Queries a flight search API with parsed parameters (origin, destination, dates, cabin class).
  • filter_results: Applies user preference constraints (max stops, price ceiling, airline preferences).
  • get_fare_details: Fetches detailed pricing for a specific flight option.
  • confirm_booking: Initiates the booking flow once the user selects a flight.

The conversation manager maintains context across the session so the user can refine their request ("actually, can you find something earlier in the morning?") without starting over.

Status

The core agent loop, flight search integration, and preference filtering are working. Current focus is on improving handling of ambiguous inputs, adding multi-city trip support, and building out a lightweight web UI for demos.
