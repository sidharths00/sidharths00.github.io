TBlocks

Block Programming App for Trashbot Robots

React Web Bluetooth API Node.js K-12 Education

Overview

TBlocks is the custom visual programming language built for Trashbots — the educational robotics platform I co-founded. Inspired by Scratch and Blockly, TBlocks lets K–12 students program physical Trashbot robots using a drag-and-drop block interface, without needing to write a single line of code.

The app runs entirely in the browser and communicates with the robot hardware over Bluetooth using the Web Bluetooth API — no app installs, no drivers, just a Chromebook and a robot. This was a critical design constraint: schools needed a zero-friction setup.

How It Works

Students drag blocks from a palette onto a canvas — blocks like "move forward", "spin motor", "sense light", and "if obstacle then stop". The block program is compiled in the browser to a bytecode representation, which is then transmitted to the Trashbot microcontroller via BLE (Bluetooth Low Energy).

The curriculum integrates with Trashbots' lesson plans across grades 2–12, scaling from basic sequences for younger students to loops, conditionals, and sensor-based logic for middle and high schoolers.

Impact

TBlocks is deployed across 100+ schools and has been used by tens of thousands of students. It's a core part of Trashbots' $400K+ in product sales and was key to winning partnerships with school districts across Texas and beyond.
