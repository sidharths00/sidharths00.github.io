TreeTies

Campus Events Finder for Stanford Students

React Node.js Bubble Web Scraping

Overview

Stanford has hundreds of events happening every week — club meetings, speaker series, free food, performances, hackathons — scattered across dozens of different mailing lists, websites, and social channels. TreeTies aggregates them all into one discoverable feed.

The platform combines two sources: automated web scraping of official Stanford event pages (the Stanford Events Calendar, department listings, and student org sites), and a community contribution layer where students can post events that the scrapers might miss.

Features

  • Feed view: A chronological list of upcoming events with filters by category, date, and location.
  • Map view: Events plotted on an interactive campus map — useful for finding what's happening nearby right now.
  • Community posts: Any Stanford student can submit an event. Submissions go through a lightweight moderation queue.
  • Save & RSVP: Users can save events to their personal calendar and get reminder notifications.

Technical Approach

The scraper runs on a Node.js backend, using a scheduled job to crawl and parse target pages. Data is cleaned and deduplicated before being stored. The frontend is a React app with a responsive two-panel layout. An early prototype was built in Bubble for rapid validation before moving to a custom stack.
